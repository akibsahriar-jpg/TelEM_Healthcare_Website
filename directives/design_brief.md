# TelEM Healthcare Website - Design Brief & Requirements

## 1. Project Context
**Goal:** Build v1 of a modern healthcare technology website for a company providing contactless, radar-based remote monitoring services.
**Audience:** B2B - Skilled Nursing Facilities, Senior Living, Hospital Operators.
**Tone:** Clinical-grade, modern, confident, quiet innovation. NOT "healthcare blue" or generic stock photos.

## 2. Technical Decisions (Clarified)
- **Framework:** **Next.js + Tailwind CSS** (via `ui-ux-pro-max` recommendations).
- **CMS Strategy:** **MDX** for Blog/Resources in v1 (keeps content in codebase but managed easily).
- **Assets:** **Programmatic Generation**. No external assets provided yet. We will use CSS/SVG animations for radar waves, signal nodes, etc.
- **Logo:** User has a placeholder logo file (needs to be provided).
- **Copy:** Agent (Designer) will write "confident, measured" draft copy.
- **Forms:** Webhook integration (console log or dummy endpoint for v1 testing).

## 3. Core Design Dimensions

### Pattern & Layout
- **Global Nav:** Sticky, blur effect. Links: Services, Resources, About, Blog. CTA: "Book a Demo".
- **Home Page Flow:**
    1.  **Hero:** "Proactive Remote Monitoring..." (Radar/Waveform visuals).
    2.  **Credibility:** Clinical oversight, compliance.
    3.  **Outcomes:** Readmission reduction, early detection.
    4.  **How It Works:** Scrollytelling (Sensor -> Trend -> Team -> Intervention).
    5.  **Services:** RPM, EM-led monitoring, Partnership model.
    6.  **Settings:** SNFs, Senior Living, etc.
    7.  **Resources & Footer.**

### Style & Aesthetic
- **Vibe:** "Healthcare intelligence software meets modern design studio."
- **Visuals:** Minimalist futurism. Strong grid, hairline borders, soft card edges.
- **Avoid:** Generic blue, stock photos of doctors shaking hands.

### Color Palette
- **Base:** Black (#070A08), Deep Charcoal (#0E1411), Soft White (#F6F8F7), Pure White (#FFFFFF).
- **Primary:** Evergreen (#0E7A4F), Emerald (#17A36B).
- **Accent:** Lime (#B9F24A) - *Limited use*.
- **Rules:** White dominates. Green guides. Black anchors.

### Typography
- **Headings:** Space Grotesk (or Sora).
- **Body:** Inter.
- **Style:** Large confident headings, readable body. Tighter letter spacing on heads.

### Animation & Interaction
- **Feel:** Subtle, purposeful, calm.
- **Key Interactions:**
    - Radar-wave background (ambient).
    - Scroll-triggered soft fades.
    - Micro-lift on buttons.

## 4. Pages (v1 Scope)
1.  **Home**
2.  **About** (Mission, Clinical Leadership)
3.  **Services / Solutions** (RPM, Readmission Reduction)
4.  **Resources** (Research, Insights)
5.  **FAQ** (Product, Privacy, Implementation)
6.  **Blog** (Educational Content - MDX)
7.  **Contact** (Lead capture)

## 5. Original Prompt Reference
> "Position the company as a credible, modern, and differentiated healthcare technology partner... The site should feel more modern than traditional healthcare sites, less flashy than consumer startups."
